基本概念

dp
device pixels 物理分辨率
px
pixels 浏览器抽象分辨率
dpr
devicePixelRatio dp/px 缩放比
ppi
pixels per inch 每英寸像素数量, 像素密度

不同像素密度的对应的缩放比

ppi 120 160 240 320
dpr 0.75 1 1.5 2

抽象分辨率推导

比如手机物理分辨率为1136dp*640dp =>
若手机是4英寸屏幕则可以求得Math.sqrt(1136*1136+640*640)/4 = 326ppi=>
326ppi的屏幕默认缩放比dpr = 2比较合适 =>
浏览器的抽象分辨率是 568px*320px
